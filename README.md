# Supermarket Sales Analysis Dashboard (Power BI)

##  Project Overview

This project presents an interactive Power BI dashboard designed to analyze supermarket sales performance. The dashboard provides insights into sales trends, regional performance, state-wise analysis, category performance, and sub-category contributions to help businesses make data-driven decisions.

---

##  Objectives

- Analyze overall sales performance.
- Identify top-performing regions and states.
- Evaluate category and sub-category sales.
- Track key business KPIs.
- Discover trends and opportunities for growth.

---

##  Dashboard Pages

### 1. Overall Sales
- Total Sales
- Total Profit
- Sales Trends
- Key Performance Indicators (KPIs)
- Overall Business Performance

### 2. Region and States Analysis
- Regional Sales Distribution
- State-wise Performance
- Profit Comparison by Region
- Geographic Insights

### 3. Category and Sub-Category Analysis
- Category-wise Sales
- Sub-Category Performance
- Top Performing Products
- Profitability Analysis

---

##  Tools Used

- Power BI Desktop
- Microsoft Excel / CSV Dataset
- Power Query
- DAX (Data Analysis Expressions)

---

##  Key Insights

- Identified top-performing regions and states.
- Compared category and sub-category contributions.
- Evaluated sales and profit trends.
- Created interactive visualizations for business reporting.
